## MAGI Supercomputer System: Operating Instructions

---

### 1. Initiating Resolution (Voting)
To trigger a system-wide resolution query between the three bio-computer cores (MELCHIOR-1, BALTHASAR-2, and CASPER-3), simply **Press the Spacebar** or **Click anywhere** on the empty space of the terminal interface.

---

### 2. System Override (Text Editing)
The MAGI terminal allows for dynamic text replacement for presentation purposes. You may **click directly on the text** of the following elements to type and overwrite them:

*   Top Banners (Default: 提訴 / 決議)
*   Center Logo (Default: MAGI)
*   Computer Designations (Default: MELCHIOR, BALTHASAR, CASPER)

---

### 3. Parameter Configuration
Access the sidebar menu to adjust the underlying system parameters:

*   **Volume (%):** Modifies the probability of a "Reject" vote. A higher volume increases the chance of rejection.
*   **Text Accept/Reject:** Click these values to input custom strings for the final voting flash sequence (e.g., changing them from 承認/否定 to YES/NO).
*   **EX_MODE:** Toggles hardware acceleration. Engaging EX_MODE significantly increases the strobe/flash speed during the voting sequence.
